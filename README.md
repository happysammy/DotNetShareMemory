# DotNetShareMemory
在.net环境中，进程间共享内存的框架

# Dev Environment
1. Visual Studio 2019
2. .Net Framework 4.0

# Quick Start
1. `using System.IO;`
2. `ShareMemory` 类可以直接调用 `Read` 和 `Write` 函数
